# Start Here

1. Install Python 3.11 or newer.
2. Open a terminal in this folder.
3. Run `setup.ps1` on Windows or `bash setup.sh` on macOS/Linux, or use the manual commands from `README.md`.
4. Create an admin user with `python manage.py createsuperuser`.
5. Visit `http://localhost:8000`.

The app works with seeded local data immediately. Real-time APIs are optional and start working after keys are added to `.env`.
